How to code in HTML

How to start your coding

If you want to start coding in HTML, then you have to put the tag <HTML> on the first line of your code. Then, you type <Head> on the next line and <Title> next to it. After that, you write </Title> with </Head>. Finally, you write your desired title in between the opening and closing tags.

How to get the main body of your text

When you have got your header in place, you have to type <Body> then write the main body of text before writing- you guessed it- </Body>. Once you have done that, just write what you want to write between the body tags. If you want a new paragraph, just enter <P/>. If you want to write out an HTML tag without its effects appearing on the document, you need an ampersand (&). What you do is you have to use the special symbols' (< & >) code and a semicolon (;). It looks like this-

&lt;Body&gt;

for the <Body> tag. Just follow that format for anything.

A new heading

If you want to get a new heading, simply do the <h1> tag but don't forget to close it! If you want a subtitle, just enter <H2> or <h3> at the end of your document.

Ending your coding

To end your coding, all you have to do is type </Body> and </HTML>. After this, the document is no longer reading your text as HTML and will ignore everything as it has no language to follow.
